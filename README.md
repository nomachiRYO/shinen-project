## このリポジトリの構成

```
site/   診断サイト本体（公開対象）
  index.html         診断アプリ。RESULTS・TYPE_IMAGES の正本
  result_*.html      16タイプの結果ページ（XLKB〜NORF）

```

## 最新の重要事項

- **タイプ名称は対応表で確定済み**（7タイプが改名）。旧名称は使用しない。
- **相性は新ロジック（L/O翻訳補完型）で確定**。docs/相性正本_16タイプ.md が唯一の拠り所。
- 各章の相性欄・各結果ページの相性は、すべて相性正本に従う。
